# CodeLikePro 🚀

> A modern Full Stack LMS Platform built with the MERN Stack to help developers learn, practice, and grow like a pro.

## 📌 Overview

**CodeLikePro** is a scalable Learning Management System (LMS) designed for students, developers, and creators.  
The platform focuses on providing a smooth learning experience with secure authentication, modern UI, and future-ready architecture.

Built using the **MERN Stack** with a clean and responsive frontend powered by **React** and **Tailwind CSS**.

---

# 🛠️ Tech Stack

## Frontend
- ⚛️ React.js
- 🎨 Tailwind CSS
- 🔥 Axios
- 🌐 React Router DOM

## Backend
- 🟢 Node.js
- 🚂 Express.js

## Database
- 🍃 MongoDB
- 📦 Mongoose

## Authentication & Security
- 🔐 JWT Authentication
- 📧 OTP-based Password Reset
- 🔑 Google OAuth Signup/Login
- 🍪 Secure Cookies & Session Handling
